Basics

print() power moves

sep / end / flush / file parameters, and debug prints

import sys, time
print("A","B","C", sep="-")             # A-B-C
print("no newline", end=""); print("...")
print("progress", end="", flush=True); time.sleep(0.1)

Functions

functools.lru_cache

Memoize pure functions for speed

from functools import lru_cache
@lru_cache(maxsize=128)
def fib(n):
    return n if n<2 else fib(n-1)+fib(n-2)
print(fib(20))
6765
Notes: Use lru_cache for expensive recursive calls.

Packaging

PyInstaller — make executables

Bundle your Python app into a single EXE/app (Windows/macOS/Linux).

# 1) Install inside a fresh virtual env
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
pip install --upgrade pip pyinstaller

# 2) Run for a simple script (one-file build)
pyinstaller --onefile app.py

# Common options
# --noconsole (GUI apps), --icon=app.ico (or .icns on macOS)
# --name MyApp, --hidden-import pkg.sub, --add-data

# Add data files (NOTE: different separators)
#   Windows -> "SRC;DEST"    macOS/Linux -> "SRC:DEST"
# Example: include a 'assets' folder into 'assets' inside the bundle
# Windows:
pyinstaller --onefile --add-data "assets;assets" app.py
# macOS/Linux:
pyinstaller --onefile --add-data "assets:assets" app.py

# 3) Find your build
#   dist/MyApp (folder) or dist/MyApp.exe (onefile .exe)
Notes: If antivirus flags the EXE, sign it or zip it. For Tkinter/turtle, prefer --onefile for small apps; test on a clean machine.
