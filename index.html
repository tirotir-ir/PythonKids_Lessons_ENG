Basics

Variable & print

Store values and print multiple items.

x = 5
y = 'hello'
print(x, y)
Basics

Simple input

Read a value from user.

name = input('Name? ')
print('Hello', name)
Strings

Concat & repeat

Use + and * on strings.

s = 'py'
print(s + 'thon')
print(s * 3)
List

Slice & append

Slice and append items.

a = [1,2,3,4]
print(a[1:3])
a.append(5)
Dict

Lookup & add

Key/Value store.

d = {'ali':12}
d['sara'] = 11
print(d['ali'])
Set

No duplicates

Unique items only.

s = {1,2,2,3}
s.add(4)
print(len(s))
Tuple

Immutable pair

Fixed position.

pt = (3,4)
print(pt[0])
Condition

if/elif/else

Three-way condition.

x=7
if x>10: print('big')
elif x==7: print('lucky')
else: print('small')
Loops

for & while

Two loop styles.

for i in range(3): print(i)

j=3
while j>0:
    print(j); j-=1
Function

def & return

Functions with outputs.

def add(a,b):
    return a+b
print(add(2,3))
List Comp.

Quick transform

Build a new list.

nums=[1,2,3]
sq=[x*x for x in nums]
print(sq)
Exceptions

try/except

Handle errors.

try:
    a=int('12a')
except ValueError:
    print('bad input')
Files

Write/Read

Simple file I/O.

open('t.txt','w',encoding='utf-8').write('hello')
print(open('t.txt','r',encoding='utf-8').read())
Modules

math / random

Useful libraries.

import math, random
print(math.sqrt(16))
print(random.randint(1,6))
Time

sleep

Pause execution.

import time
print('3...'); time.sleep(1)
print('Go!')
OOP

Simple class

Class and method.

class Dog:
    def __init__(self,n): self.n=n
    def bark(self): print('woof', self.n)
Dog('Rex').bark()
Strings

f-strings

Readable formatting.

name='Sara'
age=11
print(f'{name} is {age}')
Tools

enumerate

Index your loop.

items=['a','b']
for i,x in enumerate(items,1):
    print(i,x)
Tools

zip

Parallel iteration.

a=[1,2]; b=['one','two']
for x,y in zip(a,b):
    print(x,y)
Tools

map/filter

Transform & filter.

nums=[1,2,3,4]
evens=list(filter(lambda x:x%2==0, nums))
print(evens)

Control Flow

While loop

Repeat a block while a condition is True. Make sure the condition changes!

n = 5
while n > 0:
    print(n)
    n -= 1
print("Blast off!")
Basics

print() tips & types

Control separators and line endings; inspect variable types.

print("A","B","C", sep="-")      # A-B-C
print("no newline...", end="")
print(" then newline")
x = 3.14
print(type(x))               # <class 'float'>
print(isinstance(x, float))  # True
Modules

colorama — colored terminal

Windows-friendly ANSI colors. pip install colorama

from colorama import init, Fore, Style
init(autoreset=True)
print(Fore.GREEN + "OK")
print(Fore.RED + Style.BRIGHT + "ERROR")
Modules

termcolor — quick colors

Minimal coloring helpers. pip install termcolor

from termcolor import colored, cprint
print(colored("hello", "yellow", "on_blue", attrs=["bold"]))
cprint("warning!", "red")
Modules

pyttsx3 — text to speech (offline)

Works offline on most desktops. pip install pyttsx3

import pyttsx3
engine = pyttsx3.init()
engine.say("Hello from Python!")
engine.runAndWait()
Modules

turtle — simple graphics

Great for kids: forward/turn drawing. (Needs GUI environment)

import turtle as t
for _ in range(4):
    t.forward(80); t.left(90)
t.done()
Modules

tkinter — tiny GUI

Built-in GUI toolkit. (Needs desktop environment)

import tkinter as tk
root = tk.Tk()
tk.Label(root, text="Hello, GUI!").pack()
tk.Button(root, text="Close", command=root.destroy).pack()
root.mainloop()
